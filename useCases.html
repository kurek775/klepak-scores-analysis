UC_1 Nahrání výsledků do systému

Vstupní podmínky

  • Uživatel je přihlášený
  • Uživatel má zobrazenou obrazovku userDashboard
  1. Uživatel klikne na tlačítko +, které jej přenese na obrazovku uploadPhoto
  2. Uživatel klikne na tlačítko "Vybrat soubor" a vybere fotku s výsledky
  3. Systém fotku posílá třetí straně
  4. Systém zpracuje odpověď třetí strany
  5. Systém vrací uživateli editovatelnou tabulku s výsledky
  6. Uživatel zkontroluje výsledky případně je upraví
  7. Uživatel potvrzuje výsledky a systém je následně uloží

UC_2 Zobrazení a editace výsledků oddílu

Vstupní podmínky

  • Uživatel je přihlášený
  • Uživatel má zobrazenou obrazovku userDashboard
  1. Uživatel vybírá sport pro, který chce zobrazit či editovat výsledky
  2. Systém zobrazí výsledky
  3. Uživatel může výsledky editovat
  4. Uživatel potvrdí změny a systém je následně uloží

UC_3 Vytvoření turnusu

Vstupní podmínky

  • Admin je přihlášený
  • Admin má zobrazenou obrazovku adminDashboard
  1. Admin vyplní rok, téma turnusu
  2. Admin vybere stávající sporty nebo vytvoří nové sporty, které se budou v turnusu provozovat
  3. Admin nahraje csv soubor ve formátu 'jmeno','prijmeni','trida?','role?','oddil' s osazenstvem turnusu
  4. Admin zkontroluje formulář a potvrdí jej
  5. Systém turnus uloží

UC_4 Vytvoření a editace nové šablony

Vstupní podmínky

  • Admin je přihlášený
  • Admin má zobrazenou obrazovku adminDashboard
  1. Admin klikne na tlačítko vytvoř šablonu
  2. Admin vybere šablonu a klikne na editovat
  3. Admin je přesměrován na obrazovku templateCreator
  4. Admin vidí PDF na pravé straně obrazovky a může si jej kdykoliv v průběhu UC stáhnout
  5. Admin vyplní textové hodnoty
  6. Admin vyplní nastyluje šablonu
  7. Admin potvrdí změny
  8. Systém uloží změny

UC_5 Kontrola výsledků a generace diplomů

Vstupní podmínky

  • Admin je přihlášený
  • Admin má zobrazenou obrazovku adminDashboard
  1. Admin klikne na tlačítko GENERUJ
  2. Systém nabídne v infoDialogu možnost kontroly případné editace výsledků
  3. Pokud admin odmítne systém vygeneruje diplomy a stáhne je na PC
  4. Pokud admin s kontrolou souhlasí zobrazí se mu obrazovka resultsChecker, kde výsledky hodné diplomů uvidí a může je editovat
